
# Description

The [getdogs.py](getdogs.py) script downloads all dogs of Amazon and stores them in the [dogs/](dogs/) folder.

# How to run

For now, you only need to download the [getdogs.py](getdogs.py) file and run it with:

```bash
python getdogs.py
```
